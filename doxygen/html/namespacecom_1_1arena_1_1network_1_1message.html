Detailed Description

This package contains classes related to the network messaging system used in the Arena game framework.

The main class, com.arena.network.message.Message, represents a network message that encapsulates an action performed by a player or entity in the game. Each message includes:

  • A unique identifier (UUID) to identify the sender or the entity involved.
  • An com.arena.player.ActionEnum specifying the type of action.
  • A com.arena.game.GameNameEnum indicating the game context.
  • A timestamp representing when the action occurred, allowing message ordering.
  • A reference to a com.arena.game.entity.LivingEntity, which represents the game entity involved in the action.

Messages are comparable based on their timestamps to enable chronological sorting and processing in the networking logic.

The package supports serialization and deserialization of messages using Gson for JSON conversion.
