Detailed Description

Provides utility classes and interfaces for JSON serialization and deserialization, mainly using the Gson library.

This package includes:

  • com.arena.utils.json.GsonWorker: A concrete implementation of com.arena.utils.json.IJson, using a configured com.google.gson.GsonBuilder to enable polymorphic serialization of game entities.
  • com.arena.utils.json.IJson: An interface defining the basic methods for JSON transformation.
  • com.arena.utils.json.JsonService: A static service that centralizes access to JSON serialization methods across the application.

This system enables safe and type-aware serialization of game entities such as com.arena.game.entity.champion.Garen, com.arena.game.entity.building.Tower, etc., by supporting dynamic typing through a RuntimeTypeAdapterFactory.
It also includes a custom validation method via com.arena.utils.json.GsonWorker#validateJson(String, Class) to detect unexpected fields or unknown enum values during deserialization.
