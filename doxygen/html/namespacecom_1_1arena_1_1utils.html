Detailed Description

Provides utility classes used across the Arena project.

This package includes general-purpose helper classes such as:

  • com.arena.utils.TimeUtil: Provides time-related utilities, including timestamp generation and caller trace logging.
  • com.arena.utils.Vector2f: A lightweight 2D vector class with basic arithmetic and geometric operations like subtraction, length, distance, dot product, and directional conversion.
  • com.arena.utils.Vector3f: A lightweight 3D vector class used to represent points or directions in 3D space.

These utilities are designed to be simple, lightweight, and widely reusable in both game logic and debugging contexts.
