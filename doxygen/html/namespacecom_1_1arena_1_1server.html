Detailed Description

The com.arena.server package is responsible for managing server-side logic of the multiplayer arena game. It handles player registration, game creation and destruction, and the management of game states and entities within those games.

Core responsibilities include:

  • Maintaining a singleton com.arena.server.Server instance
  • Creating and closing game sessions identified by com.arena.game.GameNameEnum
  • Registering and retrieving com.arena.player.Player instances
  • Subscribing players to games and updating their states
  • Associating entities such as com.arena.game.entity.building.Tower, com.arena.game.entity.building.Inhibitor, and com.arena.game.entity.building.Nexus with game instances and positions using com.arena.game.utils.EntityInit
  • Logging important server events using the com.arena.utils.logger.Logger utility

This package is the backbone of the multiplayer game environment, ensuring data integrity, consistent state management, and synchronized player-game interaction.
