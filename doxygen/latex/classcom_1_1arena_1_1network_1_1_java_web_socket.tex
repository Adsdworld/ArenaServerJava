\doxysubsection{Detailed Description}
\doxylink{classcom_1_1arena_1_1network_1_1_java_web_socket}{Java\+Web\+Socket} is a singleton Web\+Socket server that listens for incoming connections and processes messages in JSON format. It uses the \doxylink{classcom_1_1arena_1_1utils_1_1json_1_1_json_service}{Json\+Service} for JSON serialization/deserialization.
