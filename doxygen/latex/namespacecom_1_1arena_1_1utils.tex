\doxysubsection{Detailed Description}
Provides utility classes used across the Arena project. 

This package includes general-\/purpose helper classes such as\+: 
\begin{DoxyItemize}
\item {\ttfamily \doxylink{classcom_1_1arena_1_1utils_1_1_time_util}{com.\+arena.\+utils.\+Time\+Util}}\+: Provides time-\/related utilities, including timestamp generation and caller trace logging. 
\item {\ttfamily \doxylink{classcom_1_1arena_1_1utils_1_1_vector2f}{com.\+arena.\+utils.\+Vector2f}}\+: A lightweight 2D vector class with basic arithmetic and geometric operations like subtraction, length, distance, dot product, and directional conversion. 
\item {\ttfamily \doxylink{classcom_1_1arena_1_1utils_1_1_vector3f}{com.\+arena.\+utils.\+Vector3f}}\+: A lightweight 3D vector class used to represent points or directions in 3D space. 
\end{DoxyItemize}

These utilities are designed to be simple, lightweight, and widely reusable in both game logic and debugging contexts.
